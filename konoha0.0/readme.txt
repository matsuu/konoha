Thank you for downloading konoha!
                              by Kimio Kuramitsu

Here is a brief instruction of enjoying konoha.

1. build

To make konoha

$ cd build
$ make

if you need to add some libraries, do as follows.

$ make LDLIBS="-ldl -lm"

2. dist/sample

Konoha is mostly like Java and in part Python. 
We have an increasing number of samples for you.

For further informaton, you will visit:
	http://konoha.sourceforge.jp/


3. Join our community

You are very welcome to our mailing list. 

http://groups.google.co.jp/group/konoha-jp-users
